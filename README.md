# 📚 计算机三级数据库技术复习题库

一个基于 Node.js + MySQL 的计算机三级数据库技术在线复习系统。

本项目重构原 JavaScript 内存题库，
使用 MySQL 数据库存储选择题数据，
实现题库持久化管理。

## ✨ Features

- ✅ 数据库存储题库
- ✅ 在线选择题练习
- ✅ 自动随机抽题
- ✅ 答题结果统计
- ✅ 错题整理（开发中）
- ✅ 题库扩展支持

## 🛠 技术栈

Frontend:
- HTML/CSS/JavaScript

Backend:
- Node.js
- Express
- MySQL2

Database:
- MySQL

## 🚀 Quick Start

1. Clone

git clone https://github.com/3541319121-code/db-review-website.git

2. Install

npm install

3. Configure MySQL

创建数据库:

db_review

导入:

question.sql

4. Run

node app.js
